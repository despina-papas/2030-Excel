An dieser Station können Sie lernen, wie Sie eine professionelle Kostenzusammenstellung in Excel erstellen. Sie wählen selbst, womit Sie starten möchten: Daten eingeben und visuell ansprechend gestalten, Daten sauber strukturieren, Berechnungen ausführen oder Kosten-Szenarien vergleichen.

Jede Themen-Seite bietet Ihnen ein klares Ziel, kurze Anweisungen, drei Schwierigkeitsstufen, kurze Video-Hilfen sowie ein kurzes Feedback und eine Reflexion am Schluss.

QR-Code Grün

Der QR-Code führt direkt zu Einsteiger-Aufträgen. Technisch kann er auf eine URL wie ?level=gruen zeigen.

QR-Code Gelb

Der QR-Code führt direkt zu mittleren Aufträgen. Technisch kann er auf eine URL wie ?level=gelb zeigen.

QR-Code Rot

Der QR-Code führt direkt zu schwierigen Aufträgen. Technisch kann er auf eine URL wie ?level=rot zeigen.

Zusätzliche Idee

Ziele

Kurzauftrag
